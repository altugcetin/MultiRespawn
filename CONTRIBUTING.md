## How to Contribute

### Reporting Bugs

1. Check if the bug has already been reported in the [Issues](https://github.com/altugcetin/FoliaRespawn/issues)
2. If not, create a new issue with:
   - A clear and descriptive title
   - Steps to reproduce the bug
   - Expected behavior
   - Actual behavior
   - Server version (Folia/Paper version)
   - Plugin version
   - Any relevant logs or screenshots

### Suggesting Features

1. Open an issue with the `enhancement` label
2. Describe the feature clearly
3. Explain why this feature would be useful

### Pull Requests

1. Fork the repository
2. Create a new branch for your feature: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Test your changes thoroughly
5. Commit with clear messages: `git commit -m "Add: feature description"`
6. Push to your fork: `git push origin feature/your-feature-name`
7. Open a Pull Request

### Code Style

- Use 4 spaces for indentation
- Follow Java naming conventions
- Add JavaDoc comments for public methods
- Keep methods small and focused

## Development Setup

1. Clone the repository
2. Import as a Gradle project in your IDE
3. Run `./gradlew build` to build the plugin
